Java Programming Language

History | Features | Usage

History

Java was developed by James Gosling at Sun Microsystems in 1995. It is a high-level, class-based, object-oriented programming language.

Initially called Oak, renamed to Java in 1995.

Features

  • Platform independent
  • Object-oriented
  • Secure
  • Robust

Java runs on JVM (Java Virtual Machine).

Usage

Java is widely used for web development, mobile apps, desktop applications, and enterprise software.
